# Victorian Plumbing Task

## Overview

This project is a small e-commerce style frontend built using **React, TypeScript, Vite, and Tailwind CSS**.  
It consumes a provided public API to fetch listing data and presents it visually

## Tech Stack

- **React** – UI rendering
- **TypeScript** – type safety and maintainability
- **Vite** – fast development and build tooling
- **Tailwind CSS** – utility first styling
- **Fetch API** – data retrieval

---

## Project Structure

```txt
src/
├── api/            # API client & request logic
├── components/     # Reusable UI components
├── hooks/          # Custom hooks
├── types/          # Shared TypeScript types
├── App.tsx         # Root UI composition
└── main.tsx        # Application entry point
```
